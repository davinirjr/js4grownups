Variables

* There's always an object called window

* This object represents the browser window.
For example, it knows how to scroll:

* Without "var", variables end up on that window object

* Stuff that lives on window are called "global".
If it's global, you don't need to say "window."

* alert is a global function

* When defined outside of a function, a variable will be global
even if you use "var"

* Without "var", the variable is global
even if you define it inside of a function.

* When a variable is not global,
we say it's local.
